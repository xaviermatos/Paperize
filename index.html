About Paperize


What is Paperize?

The Paperize project allows for all users to generate all websites into a different look which is reminecent of an old paper written from a time long ago that has now yellowed due to time.

Paperize also allows users to print websites and allow for these websites to keep the same old school paper look as shown in the example.


Why was Paperize created?

This is an open source project created by Xavier Matos as a demonstration to see how websites would look like if they were created on an old school paper in a world before the world wide web.


When will Paperize be available?

Paperize is a demo project concept that is not currently in development at this moment and was thought up for a theoretical project I could create.
